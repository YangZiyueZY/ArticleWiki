## 使用建议

1. 选择与论点高度相关的人物素材
2. 注意素材的时效性，适当使用当代人物
3. 人物事迹要准确，不可道听途说
4. 结合具体细节，增强说服力
5. 一个人物可从多个角度解读，灵活使用
