# 议论文素材库

专注于议论文素材收集、分类、整理和检索的文档系统。

## 功能特性

- 六大素材分类：人物素材、事件素材、名言警句、历史典故、科学研究、社会热点
- 完善的标签管理体系，支持精准检索
- 本地搜索功能，快速定位素材
- 收藏功能，建立个人专属素材库
- 响应式设计，支持亮/暗色模式

## 快速开始

```bash
# 安装依赖
yarn install

# 启动开发服务器
yarn docs:dev

# 构建生产版本
yarn docs:build

# 预览生产构建
yarn docs:preview
```

## 素材分类

| 分类 | 描述 |
|------|------|
| 人物素材 | 古今中外典型人物事迹 |
| 事件素材 | 时事新闻与社会热点事件 |
| 名言警句 | 经典名言与格言警句 |
| 历史典故 | 历史故事与经典典故 |
| 科学研究 | 科学发现与研究数据 |
| 社会热点 | 当下热门社会话题 |
